# Xiaomi Band Input Method / 小米手环输入法

Input texts in **Chinese**, **English**, and **Japanese** on Xiaomi Band.  
In fact, more languages can also be supported by revising the corresponding `.ux` files.

可在 **小米手环** 上输入 **中文、英文、日文**。  
实际上，只要修改对应的 `.ux` 文件，也可以继续扩展支持更多语言。

---

## Features / 功能说明

Based on **NEORUAA's** open-source project, this version adds:

- Around **2,000** more common Japanese kanji
- Some commonly used punctuation marks

基于 **NEORUAA** 的开源项目，本版本新增了：

- 约 **2000** 个常用日语汉字
- 一些常用标点符号

---

## Tips / 使用提示

### 1. Yōon / 拗音

Yōon should be split into **two characters**.

拗音需要拆分成 **两个字符** 输入。

**Example / 示例：**

- `りょ (ryo)` → `ri (り)` + `lyo (ょ)`

---

### 2. Small Kana / 小假名

Use `l` + vowel to input small kana.

使用 `l + 元音` 输入小假名。

**Example / 示例：**

- `l` + `a` → `ぁ`

Other small kana follow the same rule.

其他小假名同理。

---

### 3. Sokuon / 促音

**Example / 示例：**

- `tt` → `っ`

---

### 4. Hatsuon / 拨音

**Example / 示例：**

- `nn` → `ん`

---

## Notes / 备注

If you want to support more languages or symbols, you can extend the input rules and revise the related `.ux` files accordingly.

如果你想支持更多语言或符号，可以继续扩展输入规则，并修改相关 `.ux` 文件。
